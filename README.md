# Face Recognition Attendance System

An AI-powered attendance tracking system that uses facial recognition to automate student attendance.

## About
Developed using Python and facial recognition algorithms to apply computer vision concepts 
and AI fundamentals in a functional prototype.

## Pipeline
Image -> Face Detection -> Cropped Face -> Face Recognition -> Name -> Mark Attendance

## Dependencies
- Python 3.4+
- TensorFlow
- OpenCV
